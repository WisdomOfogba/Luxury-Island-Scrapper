# Luxury Island Scraper

## The Geographic Analysis (Wisdom Ofogba)
This shows the **Top 10 Business Location** and **The Locations with the highest minimum entry price *(Eclusive Chart)***

### Packages to be installed
- pandas
- requests
- seaborn
- matplotlib

### Incase you dont have them istalles yet
- clone this repo and run this on your *CLI*
```
pip install -r requirements.txt
```
- Open the `geographic_analysis.ipynb` file and run the notebook
